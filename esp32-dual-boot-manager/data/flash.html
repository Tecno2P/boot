How to Flash Firmware
  1. Compile your firmware in Arduino IDE: Sketch → Export Compiled Binary
  2. Find the .bin file in your sketch folder
  3. Select the target slot (APP0 or APP1) — note the size limit for each
  4. Click Flash and wait for the upload to complete
  5. After success, click Set Boot + Reboot to switch to the new firmware
